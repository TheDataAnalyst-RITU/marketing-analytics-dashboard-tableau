# Tableau-Project---Marketing-Analytics-Dashboard
It's about how marketing leaders understand cross-channel paid media performance - tracking spend, CTR, CPM, &amp; conversions across channels, platforms, &amp; campaigns. Built on synthetic 2023 data (~11.7K rows, 4 channels, 8 platforms), it surfaces the patterns analysts actually act on: channel rotation, efficiency drift, &amp; platform mix shifts.
